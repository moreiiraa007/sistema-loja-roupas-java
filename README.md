# 🛍️ Sistema de Loja de Roupas em Java

Projeto desenvolvido em Java com foco em prática de lógica de programação e construção de um sistema de vendas via terminal.

---

## 🚀 Funcionalidades

- Menu interativo no terminal
- Adição de produtos ao carrinho
- Visualização do resumo do pedido
- Cálculo automático do total da compra
- Aplicação de desconto (10% para compras acima de R$200)
- Escolha de forma de pagamento (dinheiro ou cartão)
- Cálculo de troco (para pagamento em dinheiro)
- Validação de entradas (não permite valores inválidos)

---

## 🧠 Conceitos aplicados

- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`while`)
- Métodos (funções)
- Entrada de dados com `Scanner`
- Organização e separação de responsabilidades no código

---

## 💻 Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git e GitHub

---

## 📌 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar os fundamentos da linguagem Java e evoluir na lógica de programação, simulando um sistema real de vendas.
